double mathlib::factorial ( int  n)

Implementation of factorial function.

Computes the factorial of a non-negative integer.

Iterative computation from 1 to n

Uses iterative approach instead of recursive for:

  • Better performance (no stack overhead)
  • No stack overflow risk for large n
  • Easier to optimize by compiler

Algorithm:

  1. Check for negative input (throw exception)
  2. Handle base cases (0! = 1, 1! = 1)
  3. Multiply from 2 to n iteratively
Time Complexity:
O(n) where n is the input value
Space Complexity:
O(1) - uses only constant extra space

Calculates \( n! = n \times (n-1) \times (n-2) \times \ldots \times 1 \)

The factorial function is defined as:

\[ n! = \begin{cases} 1 & \text{if } n = 0 \\ n \times (n-1)! & \text{if } n > 0 \end{cases} \]

Common applications:

  • Combinatorics: \( \binom{n}{k} = \frac{n!}{k!(n-k)!} \)
  • Probability distributions (Poisson, etc.)
  • Taylor series expansions: \( e^x = \sum_{n=0}^{\infty} \frac{x^n}{n!} \)
Parameters
nThe input non-negative integer
Returns
The factorial of n (n!)
Exceptions
std::invalid_argumentif n < 0
Example:
double result = mathlib::factorial(5); // result = 120.0
// For binomial coefficient:
int n = 10, k = 3;
double binom = factorial(n) / (factorial(k) * factorial(n - k));

double mathlib::square ( double  x)

Implementation of square function.

Computes the square of a number.

Simple multiplication: x * x

This implementation uses direct multiplication rather than pow(x, 2) for efficiency. Direct multiplication is:

  • Faster (no function call overhead)
  • More accurate (no floating-point exponentiation)
  • Compiler can optimize effectively

Calculates \( x^2 \) for a given input value.

This function is commonly used in:

  • Distance calculations: \( d = \sqrt{x^2 + y^2} \)
  • Energy computations: \( E = \frac{1}{2}mv^2 \)
  • Statistical variance calculations
Parameters
xThe input value
Returns
The square of x (x²)
Example:
double value = 5.0;
double result = mathlib::square(value); // result = 25.0
